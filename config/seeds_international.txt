# Athena — Internationale Forschungs- und Studienquellen
# ------------------------------------------------------
# Seed-URLs für rss_watch.py / crawl.py / manuelles Einreichen.
# Jede Quelle landet als Tier-0-Submission und durchläuft auto_review.py, das
# - publisher_trust prüft,
# - thematische Topics setzt,
# - country + transferability_de bewertet (siehe REVIEW_SCHEMA_HINT).
# Verifizierter Tier-Rang wird erst durch menschliche Freigabe vergeben.
#
# Format: ein URL pro Zeile, Kommentar mit "#". Leerzeilen / Sektions-Header
# erlaubt — Crawler ignoriert alles vor dem ersten "http".
#
# Auswahl-Kriterium: institutionelle Forschungsquellen mit nachvollziehbarer
# Methodik, in der jeweiligen Politikdebatte zitiert. KEINE Think-Tanks mit
# verdeckter Finanzierung, keine reinen Advocacy-Seiten.
